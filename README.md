# Linux Shell in C

Simple Linux shell.


## Built With

* C

## currently working commands:

* cd
* help
* exit


## To Run the program:

Compile shell.c from Linux terminal

```
$ gcc shell.c -o shell
```

or for better compilation,
```
$ gcc -Wall -Wextra -O2 -g -o shell shell.c
```

Execute the program

```
$ ./shell
```

You are now in the shell
